## condicionales

```python
x = 10

if x > 5:
  print("x es mayor que 5")
elif x == 5:  # máximo hasta 7
  print("x es igual a 5")
else:
  print("x es menor que 5")

match x:
  case 1:
    print("x es 1")
  case 2:
    print("x es 2")
  case _:
    print("x no es ni 1 ni 2")
```
